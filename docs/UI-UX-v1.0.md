## 1. 设计体系基础

### 1.1 设计原则

| 原则 | 说明 |
|------|------|
| 3秒法则 | 记录流程 ≤2步，核心操作一步直达 |
| 情绪优先 | 视觉层次以情绪表达为核心，数据辅助 |
| 零压力 | 柔和色调，不使用警示性颜色 |
| 隐私感 | 大圆角、大留白、柔和过渡，营造私人日记安全感 |
| 渐进披露 | 信息按优先级分层，不一次性轰炸 |

### 1.3 字体系统

| 级别 | 大小 (sp) | 字重 | 行高 | 用途 |

| H1 | 28 | Bold (700) | 36 | 页面标题（"记录情绪"） |
| H2 | 22 | Semibold (600) | 28 | 区域标题（"今日记录"） |
| H3 | 18 | Semibold (600) | 24 | 子标题、Tab 标签 |

- iOS: SF Pro（系统默认）
- Android: Roboto / Noto Sans（系统默认）
- 情绪图标: Apple Color Emoji / Noto Emoji

交互规格：

| 交互 | 规格 |
|------|------|
| 情绪选择 | 点击后图标放大1.15x + 弹跳动画（200ms spring），边框2px情绪色高亮；其余选项透明度降至0.4 |
| 强度滑块 | 默认值3，拖动时轨道左侧填充情绪色，滑块颜色跟随所选情绪色；5级离散刻度，每级间距相等 |
| 备注输入 | 点击展开输入框，键盘弹出时页面整体上推，保存按钮固定在键盘上方（sticky bottom），键盘工具栏含"完成"按钮 |
| 时间修改 | 点击顶栏时间文字→弹出系统 DateTime Picker，可选范围：当前时间往前7天 |
| 保存按钮 | 未选情绪：灰色禁用态（不可点击）；已选情绪：激活为对应情绪色 |
| 保存成功 | 选中情绪图标执行放大+粒子扩散动画（200ms内），粒子颜色为对应情绪色；动画完成后界面重置 |
| 低端机降级 | 首帧渲染超过16ms自动降级为简单缩放动画 |
| 最近记录 | 展示最近3条记录，点击跳转列表页 |
